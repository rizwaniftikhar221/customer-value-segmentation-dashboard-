# Customer Value Segmentation Dashboard – Power BI + BigQuery + SQL

## Overview

This project is a Power BI dashboard created to analyze customer transaction data and segment customers using RFM analysis. The dashboard focuses on identifying customer value, engagement levels, and potential churn risk.

The main idea of this project is to transform raw sales data into clear customer segments and support targeted business decisions using simple and interpretable metrics.

## Why This Project

In many business environments, raw sales data does not clearly show which customers are valuable or which are at risk of churn. This project converts transactional data into structured customer segments to make decision making easier.

The dashboard helps highlight high value customers, growth opportunities, and risk segments, enabling more focused retention and engagement strategies.

## What the Dashboard Shows
- Total number of customers
- Customer distribution across RFM segments
- Identification of high value customers (Champion, Loyal VIPs)
- Identification of at risk and inactive customers
- Detailed customer level metrics including recency, frequency, and monetary values

The dashboard provides a clear view of customer value distribution and supports quick identification of key segments for business action.

## Data Used
- Customer sales transaction dataset
- Monthly data consolidated into a single dataset
- Includes customer ID, order date, and order value

## Tools
- Google BigQuery for data processing
- SQL for data transformation and RFM analysis
- Power BI for dashboard development and visualization
